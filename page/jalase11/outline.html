Attribut outline:
ویژگی outline در CSS برای ترسیم یک خط در اطراف یک عنصر استفاده می‌شود که در خارج از لبه‌های آن قرار می‌گیرد. برخلاف border، outline بخشی از مدل جعبه (Box Model) نیست و بر اندازه یا مکان عنصر تأثیر نمی‌گذارد.

ویژگی‌های تشکیل‌دهنده:
1-outline-width: تعیین ضخامت خط.
مقادیر: thin، medium، thick، یا مقدار دقیق مانند 2px.
}div
outline-width: 3px;
{

2-outline-style: تعیین نوع خط.
مقادیر: solid، dotted، dashed، double، none و غیره.
} div
outline-style: dashed;
{

3-outline-color: تعیین رنگ خط.
مقادیر: color-name، hex، rgb یا currentColor.
} div
outline-color: red;
{

4-outline-offset: فاصله خط از لبه عنصر.
مقدار: اندازه‌ای مشخص مثل 5px.
} div
outline: 2px solid blue;
outline-offset: 10px;
{

5-outline (Shorthand):ترکیب تمام مقادیر بالا.
ساختار: outline: [width] [style] [color].
} div
outline: 2px solid green;
