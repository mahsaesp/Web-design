آشنایی با رنگها در css


1-نام رنگها(Color):

css از تعدادی رنگ پشتیبانی می کند که میتوانیذد به راحتی همراه با نام رنگ از انها استفاده کنید

Color:red Color:blue Color:green

2-کدهای هگزادسیمال(Hexadcimal):

کدهای هگزادسیمال با علامت شارپ (#) شروع می شوند و شامل شش رقم (یا سه رقم خلاصه شده) هستند که نشان دهنده میزان سه رنگ اصلی قرمز ،آبی و سبز است.


color:#ff000 color:#00ff00 color:#0000ff color:#333

3-رنگهای (RGB):

این مدل مقدار قرمز (R)،سبز(G)و آبی(B) را به صورت اعداد اعشاری بین 0الی 255 مشخص می کند. همچنین میتوان از مقادیر درصدی نیز استفاده کرد.


rgb(255,0,0) rgb(0,255,0) rgb(0,0,255) rgb(100%,0%,0%)

4-رنگهای RGBA:

این مدل مشابه RGB است ولی مقدار شفافیت(opacity) نیز در آن مشخص می شود.

color:rgba(255,0,0,0.5)

5-رنگهای HSL:

مدل HSL برای تعریف رنگ بر اساس تناژ(Hue)،میزان اشباع(Saturation)و رو شنایی(Lightness) استفاده می شود.

color:hsl(0,100%,50%) color:hsl(120,100%,50%) color:hsl(240,100%,50%)

6-رنگهای HSLA:

این مدل مشابه Hsl است ولی مقدار شفافیت (Alpha) نیز در آن تعریف می شود.

hsla(0,100%,50%,0.5)

7-مقادیر شفافیت(Transparency):

برای تغییر شفافیت کل یک عنصر می توان از ویژگی opacity استفاده کرد.

opacity:0.5

نکته: برای استفاده حرفه ای تر یهتر است برای انتخاب رنگ از ابزارهایی مانند Coolors و Adobe color بهره ببرید تا طراحی های هماهنگ تری ایجاد کنید.
