Tage link:

تگ <link> در HTML برای ارتباط دادن صفحه‌ی وب به منابع خارجی (مانند فایل‌های CSS، آیکون‌ها و ...) استفاده می‌شود. این تگ معمولاً در بخش <head> صفحه قرار می‌گیرد و به تنهایی قابل نمایش در صفحه نیست. تگ <link> برای اتصال منابع خارجی یا متادیتاها است و در صفحه نمایش داده نمی‌شود.

ویژگی‌های مهم تگ link:

1)rel: نوع رابطه بین صفحه و منبع موردنظر. نمونه‌هایی از مقادیر

2)stylesheet: برای اتصال فایل‌های CSS.

<link rel="stylesheet" href="styles.css">

3)icon: برای مشخص کردن آیکون وب‌سایت (Favicon)

4)href: مسیر یا آدرس منبع خارجی

5)type: نوع فایل پیوند داده‌شده (اختیاری). برای CSS، مقدار معمولاً text/css است.

<link rel="icon" href="favicon.ico" type="image/x-icon">

6)media: مشخص می‌کند که فایل موردنظر برای چه دستگاه یا سایزی مناسب است (مثلاً برای موبایل یا چاپ).

<link rel="stylesheet" href="print.css" media="print">

Tage table:

تگ <table> در HTML برای ایجاد جداول استفاده می‌شود. جداول به ما اجازه می‌دهند داده‌ها را در قالب ردیف‌ها و ستون‌ها سازمان‌دهی کنیم.

تگ‌ها و ویژگی‌های مرتبط:

1)<table>: عنصر اصلی که جدول را تعریف می‌کند.

2)<tr>: هر ردیف جدول.

3)<td>: سلول داده در یک ردیف.

4)<th>: سرستون یا سلول عنوان که معمولاً متن آن ضخیم و تراز وسط (مرکز) است.

5)<thead>: برای گروه‌بندی سرستون‌ها.

6)<tbody>: برای گروه‌بندی داده‌های جدول.

7)<tfoot>: برای گروه‌بندی بخش پایانی جدول (مثل خلاصه یا جمع‌بندی).

8)rowspan: ترکیب سلول‌ها در چند ردیف.

9)colspan: ترکیب سلول‌ها در چند ستون.

<table border="1">
<tr>
<th>name</th>
<th>Details</th>
<tr/>
<tr>
<td rowspan="2">ali</td>
<td>Student</td>
<tr/>
<tr>
<td>tehran</td>
<tr/>
<tr>
<td colspan="2">Conclusion</td>
<tr/>
<table/>

مثال کد بالا:
نام جزئیات
علی دانشجو
تهران
جمع‌بندی


Attribut scope:

در HTML، ویژگی scope در تگ <th> استفاده می‌شود تا مشخص کند که سرستون یا سرسطر یک جدول (table header) به کدام بخش از جدول اعمال می‌شود. این ویژگی به خوانایی بیشتر جداول توسط ابزارهای کمکی مانند screen readers کمک می‌کند بنابراین دسترسی‌پذیری (accessibility) جداول افزایش می‌یابد.

<th scope="row">row header</th>


Tage kbd:

تگ <kbd> در HTML برای نمایش ورودی کاربر از طریق صفحه‌کلید، صدا، یا هر وسیله ورودی دیگر استفاده می‌شود. این تگ معمولاً برای مشخص کردن کلیدها یا ترکیب کلیدهایی که کاربر باید فشار دهد به کار می‌رود و به صورت پیش‌فرض، متن داخل آن با یک فونت تک فاصله (monospace) نمایش داده می‌شود.

Tage code:

تگ <code> در HTML برای نمایش کدهای برنامه‌نویسی یا هر نوع محتوای مرتبط با کدنویسی استفاده می‌شود. این تگ متن داخل خود را با یک فونت تک‌فاصله (monospace) نمایش می‌دهد که برای نمایش قطعات کد مناسب است.


Tage samp:

<samp> در HTML برای نمایش خروجی یک برنامه، دستگاه یا سیستم استفاده می‌شود. این تگ معمولاً برای نمایش پیام‌های سیستم یا نتایج حاصل از اجرای کد کاربرد دارد و به صورت پیش‌فرض با فونت تک‌فاصله (monospace) نمایش داده می‌شود.


Tage var:

تگ <var> در HTML برای نمایش متغیرها در یک عبارت ریاضی، کد برنامه‌نویسی، یا هر نوع محتوایی که شامل یک متغیر است استفاده می‌شود. متن داخل این تگ به‌صورت پیش‌فرض با فونت ایتالیک (italic) نمایش داده می‌شود تا نشان‌دهنده‌ی متغیر بودن آن باشد.
