آشنایی با css

CSS (Cascading Style Sheets):

css یک زبان استایل‌دهی است که برای تنظیم ظاهر و طراحی صفحات وب استفاده می‌شود. این زبان به شما امکان می‌دهد تا به عناصر html ظاهر زیبا، موقعیت دلخواه و طراحی پویا بدهید.


ویژگی‌های اصلیcss:

1.جداسازی محتوا از ظاهر: CSSبه شما کمک می‌کند تا ظاهر (استایل) و محتوارا از هم جدا کنید. این امر باعث افزایش خوانایی و مدیریت بهتر کد(html)می شود.
2.واکنش‌گرایی (Responsive Design): CSS به شما امکان می‌دهد صفحات وب را برای نمایش مناسب در دستگاه‌های مختلف (موبایل، تبلت و دسکتاپ) طراحی کنید.
3.کنترل ظاهر عناصر: CSSبه شما این امکان را می‌دهد که رنگ‌ها، اندازه‌ها، فونت‌ها، موقعیت‌ها و سایر ویژگی‌های ظاهری عناصر html را تغییر دهید.
4.قابلیت استفاده مجدد: با نوشتن استایل در یک فایل خارجی، می‌توانید از همان استایل در چندین صفحه استفاده کنید.


انواع css:

ccرا می‌توان به سه روش مختلف در صفحات وب استفاده کرد:

(Internal CSS)cssداخلی: استایل ها در داخل تگ <style> و در فایل html تعریف می شود
<style>
h1 {
color:blue;
fontz-size:24px;
}
</style>

(Inline CSS)cssدرونی: به صورت مستقیم در تگ های html نوشته می شود.
</h1>سلام <h1 style="color: blue; font-size: 24px>

(External CSS)cssخارجی: استایل‌ها در یک فایل جداگانه با پسوند css.نوشته شده و به فایل html متصل می‌شوند.
<link rel="stylesheet" href="styles.css">


ساختار CSS:

ساختار CSS شامل قوانین (Rules) است که به عناصر HTML استایل می‌دهد. هر قانون CSS از انتخابگرها (Selectors) و اعلان‌ها (Declarations) تشکیل شده است.
یاختار کلی درcss:
1-انتخابگر (Selector): بخشی که عنصر یا عناصری را که باید استایل داده شوند، مشخص می‌کند .
2-ویژگی (Property): خصوصیتی از عنصر که می‌خواهید تغییر دهید (مانند رنگ، اندازه، فونت، و ...).
3-مقدار (Value): مقدار مشخص شده برای ویژگی.

خلاصه ساختار CSS

بخش توضیح مثال
انتخابگر مشخص‌کننده عنصر HTML h1, .class, #id
ویژگی (Property) تعیین‌کننده خاصیت عنصر color, font-size
مقدار (Value) مقدار خاصیت مشخص شده blue, 16px


ابزارهایی برای انتخاب عناصر HTML و اعمال استایل بر روی آن‌ها:

1. نقطه (.)

نقطه در CSS برای تعریف کلاس‌ها (Classes) استفاده می‌شود. کلاس‌ها یک نام عمومی هستند که می‌توانند به چندین عنصر HTML اختصاص داده شوند. از کلاس‌ها برای استایل‌دهی به گروهی از عناصر استفاده می‌شود.
}className.
property: value;
{

2. شارپ (#)

شارپ برای تعریف آی‌دی (ID) در CSS استفاده می‌شود. آی‌دی یک نام یکتا است که فقط به یک عنصر HTMLاختصاص داده می‌شود.
} idName#
property: value;
{

3. ستاره (*)

ستاره یک انتخابگر جهانی (Universal Selector) در CSS است که تمام عناصر صفحه را انتخاب می‌کند.
} *
property: value;
{

4. ترکیب نقطه، شارپ و ستاره

می‌توانید هم‌زمان از کلاس و ID برای استایل‌دهی استفاده کنید.

} main.content#
background-color: lightgreen;
{

تفاوت نقطه (کلاس) و شارپ (ID)

ویژگی کلاس (.) آی‌دی (#)
نحوه تعریف با نقطه (`.`) آغاز می‌شود با شارپ (`#`) آغاز می‌شود
کاربرد برای استایل‌دهی گروهی از عناصر برای شناسایی یکتای یک عنصر
تعداد استفاده می‌تواند به چند عنصر اعمال شود فقط به یک عنصر اعمال می‌شود
اولویت اولویت کمتر نسبت به ID اولویت بالاتر نسبت به کلاس


استایل دهی تو در تو

استایل‌دهی تو در تو در CSS به معنای استفاده از انتخابگرهای ترکیبی و سلسله‌مراتبی است که به شما امکان می‌دهد استایل‌ها را برای عناصر خاصی که در داخل عناصر دیگر قرار دارند، اعمال کنید. این روش به مدیریت دقیق‌تر و خوانایی بیشتر کدها کمک می‌کند.

انتخابگرهای سلسله‌مراتبی (Descendant Selectors): انتخاب عناصری که در داخل عناصر دیگر هستند.

{
