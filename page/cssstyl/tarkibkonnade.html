ترکیب‌کننده‌ها (Combinators) در CSS

ابزارهایی هستند که برای تعیین رابطه بین عناصر HTML به کار می‌روند. این ترکیب‌کننده‌ها به شما اجازه می‌دهند استایل‌ها را بر اساس روابط میان عناصر اعمال کنید.

انواع ترکیب‌کننده‌ها در CSS

1-ترکیب‌کننده فرزند مستقیم (>): فقط به فرزندان مستقیم یک عنصر اعمال می‌شود.
} div > p
;color: blue
{
تمامی پاراگراف‌هایی که مستقیماً داخل یک <div> قرار دارند، آبی می‌شوند.

2-ترکیب‌کننده فرزند (space): به همه‌ی فرزندان (مستقیم یا غیر مستقیم) یک عنصر اعمال می‌شود.
} div p
color: red;
{
همه‌ی پاراگراف‌ها داخل یک <div>، بدون توجه به سطحشان، قرمز می‌شوند.

3-ترکیب‌کننده مجاور (+): به عنصری که دقیقاً بعد از عنصر دیگر می‌آید، اعمال می‌شود.
} h1 + p
color: green;
{
فقط اولین پاراگراف بعد از هر <h1> سبز می‌شود.

4-ترکیب‌کننده عمومی (~):به همه‌ی عناصر هم‌سطح که بعد از عنصر مشخص‌شده می‌آیند، اعمال می‌شود.
} h1 ~ p
color: purple;
{
همه‌ی پاراگراف‌هایی که هم‌سطح با <h1> هستند و بعد از آن قرار دارند، بنفش می‌شوند.

5-ترکیب‌کننده گروه (,):برای استایل دادن به چندین انتخابگر مختلف.
} h1, h2, p
font-family: Arial, sans-serif;
{
تمام <h1>, <h2> و <p> از فونت Arial استفاده می‌کنند.
