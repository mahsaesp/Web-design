text-decoration:

ویژگی text-decoration در CSS برای اضافه کردن یا تغییر استایل تزئینات متن استفاده می‌شود. این ویژگی معمولاً برای مواردی مانند زیر خط بردن متن، خط زدن متن، یا حتی حذف خط زیر لینک‌ها به کار می‌رود.


مقادیر پرکاربرد text-decoration:

1.none:هیچ تزئینی روی متن اعمال نمی‌شود.

} a
/* خط زیر لینک حذف می‌شود */ ;text-decoration: none
{

2.underline: خط زیر متن.

} a
/*خط زیر متن پاراگراف*/ ;text-decoration: underline
{

3.overline:خط بالای متن.

} h
/*خط بالای عنوان*/ ;text-decoration: overline
{

4.line-through:خط روی متن (برای نمایش متن خط‌خورده).

} del
/*نمایش متن حذف‌شده*/ ;text-decoration: line-through
{

5.underline overline:خط بالا و پایین متن.

} h2
;text-decoration: underline overline
{

نکات:

برای کنترل دقیق‌تر، می‌توانید از زیرشاخه‌های این ویژگی استفاده کنید:
text-decoration-line: نوع تزئین (مثل underline، overline).
text-decoration-color: رنگ تزئین.
text-decoration-style: استایل خط (مثل solid، dashed، wavy).

text-align:

ویژگی text-align در CSS برای تعیین نحوه تراز کردن متن در داخل یک عنصر استفاده می‌شود. این ویژگی به شما امکان می‌دهد متن را به چپ، راست، وسط یا به صورت هم‌تراز (justified) تنظیم کنید.


مقادیر رایج text-align:

1.left:تراز متن به سمت چپ (حالت پیش‌فرض در زبان‌های چپ‌چین مانند انگلیسی).

} p
; text-align: left
{

2.right:تراز متن به سمت راست (مناسب برای زبان‌های راست‌چین مانند فارسی و عربی).

} p
;text-align: right
{

3.center:تراز متن به مرکز.

} h1
; text-align: center
{

4.justify:متن به صورت کامل هم‌تراز می‌شود (کشیده شدن متن از هر دو طرف).

} p
; text-align: justify
{

5.start:متن را مطابق با جهت زبان (LTR یا RTL) تراز می‌کند. برای زبان‌های چپ‌چین مثل انگلیسی، معادل left است. برای زبان‌های راست‌چین مثل فارسی، معادل right است.

} p
; text-align: start
{

6.end:متن را از انتهای زبان (LTR یا RTL) تراز می‌کند. برای زبان‌های چپ‌چین معادل right و برای زبان‌های راست‌چین معادل left است.

} p
;text-align: end
{

نکات:

برای زبان‌های راست‌چین مثل فارسی، حتماً از ویژگی direction: rtl استفاده کنید تا رفتار صحیحی داشته باشید.
ویژگی text-align فقط بر محتوای درون یک عنصر تأثیر می‌گذارد، نه خود عنصر.

line-height:

ویژگی line-height در CSS برای تنظیم فاصله بین خطوط متن در یک عنصر استفاده می‌شود. این ویژگی می‌تواند به متن ظاهر مرتب‌تری بدهد و خوانایی آن را افزایش دهد.

مقادیر قابل قبول line-height:

1.واحد عددی (number):این مقدار یک ضریب بدون واحد است که بر اندازه فونت ضرب می‌شود.

} p
; font-size: 16px
/* فاصله خطوط 1.5 برابر اندازه فونت است */;line-height: 1.5
{

2.واحد نسبی یا مطلق (px, em, %, rem):شما می‌توانید فاصله خطوط را با واحد مشخصی تنظیم کنید.

}p
/* فاصله خطوط دقیقاً 24 پیکسل است */ ;line-height: 24px
{
}h1
/* فاصله خطوط 150 درصد اندازه فونت است */ ;line-height: 150%
{

3.normal:مقدار پیش‌فرض مرورگر برای فاصله خطوط، که معمولاً بسته به فونت و اندازه متن متغیر است.

} p
;line-height: normal
{

نکات مهم:

مقدار عددی (بدون واحد) معمولاً ترجیح داده می‌شود، زیرا با تغییر اندازه فونت، فاصله خطوط نیز متناسب تغییر می‌کند.
مقدار خط باید به اندازه کافی بزرگ باشد تا متن خوانا باشد، اما نه آنقدر بزرگ که پراکندگی بیش از حد ایجاد کند.

overflow:

ویژگی overflow در CSS برای تعیین نحوه نمایش محتوا در صورتی که از محدوده عنصر (Container) خارج شود، استفاده می‌شود. این ویژگی مشخص می‌کند که آیا محتوای اضافی باید پنهان شود، اسکرول‌بار داشته باشد یا به حالت پیش‌فرض نمایش داده شود.

مقادیر قابل قبول overflow:

1.visible (پیش‌فرض):محتوا بدون هیچ محدودیتی نمایش داده می‌شود و از محدوده عنصر خارج می‌شود.

} div
; overflow: visible
{

2.hidden:محتوای اضافی مخفی می‌شود و قابل مشاهده نیست.

}div
; overflow: hidden
{

3.scroll:اسکرول‌بار افقی و عمودی اضافه می‌شود، حتی اگر محتوای اضافی وجود نداشته باشد.

} div
; overflow: scroll
{

4.auto:اسکرول‌بار فقط در صورت نیاز (زمانی که محتوا از محدوده عنصر خارج شود) اضافه می‌شود.

} div ;overflow: auto {

نکات:

overflow-y و overflow-x ویژگی است که برای کنترل نوار اسکرول‌ افقی و عمودی به طور جداگانه استفاده می شود اگر از hidden استفاده کنید محتوای اضافی از دید خارج می شود و هیچ اسکرولی ظاهر نمی شود.


max-width:

ویژگی max-width در CSS برای تعیین حداکثر عرض یک عنصر استفاده می‌شود. این ویژگی باعث می‌شود که عرض عنصر از مقدار تعیین‌شده فراتر نرود، حتی اگر محتوای داخل عنصر بزرگ‌تر باشد.


مقادیر قابل قبول max-width:

1.واحدهای طول (px, %, em, rem, vh, vw): عرض حداکثر با استفاده از واحدهای مشخص تنظیم می‌شود

}div
/* حداکثر عرض عنصر 600 پیکسل است */ ;max-width: 600px
{

2.none:هیچ محدودیتی برای عرض عنصر وجود ندارد.

}div
;max-width: none
{

3.نسبت به والد یا viewport: می‌توانید از واحدهای درصد یا نسبی استفاده کنید:

} div /* حداکثر عرض 80٪ از عرض والد */ ;max-width: 80% {

نکات:
استفاده از max-width به جای width در طراحی وب واکنش‌گرا (Responsive) توصیه می‌شود.
این ویژگی می‌تواند با min-width ترکیب شود تا محدوده عرض یک عنصر را تعیین کنید.

vertical-align:

ویژگی vertical-align در CSS برای تعیین نحوه تراز عمودی یک عنصر استفاده می‌شود. این ویژگی معمولاً برای تراز کردن متن، تصاویر یا سایر عناصر درون یک عنصر inline، inline-block، یا یک سلول جدول به کار می‌رود.

مقادیر رایج vertical-align:

1.ترازهای کلمه‌کلیدی (keyword values):
تراز کلمه‌کلیدی توضیحات
baseline تراز متن با خط پایه (پیش‌فرض)
top تراز متن با بالای خط والد
middle تراز متن با مرکز ارتفاع خط
bottom تراز متن با پایین خط والد
text-top تراز متن با بالای متن در خط والد
text-bottom تراز متن با پایین متن در خط والد
sub تراز متن به صورت زیرنویس (subscript)
super تراز متن به صورت بالانویس (superscript)

2.مقادیر عددی یا نسبی: می‌توانید از مقادیر طول (مانند px, em, %) برای تغییر موقعیت عنصر نسبت به خط پایه استفاده کنید

} span
/* 10px بالاتر از خط پایه */ ;vertical-align: 10px
{

نکات:

فقط برای عناصر inline یا table-cell کار می‌کند. اگر عنصر شما block باشد، ابتدا باید آن را به inline- block تغییر دهید
این ویژگی روی متن و عناصر inline داخل خط تأثیر دارد، اما برای تراز عمودی کل یک باکس باید از ترکیب flexbox یا grid استفاده کنید.


font-family:

ویژگی font-family در CSS برای تعیین نوع فونت متن در یک عنصر استفاده می‌شود. این ویژگی به شما امکان می‌دهد یک یا چند فونت را برای متن مشخص کنید. مرورگر ابتدا فونت اول را امتحان می‌کند و اگر در دسترس نبود، به سراغ فونت‌های بعدی می‌رود.

} selector
;font-family: "Font 1", "Font 2", generic-family
{

نکات:

همیشه از فونت‌های جایگزین (Fallback) استفاده کنید.
استفاده از فونت‌های Web Safe (مانند Arial یا Verdana) برای افزایش سازگاری توصیه می‌شود.
برای فونت‌های سفارشی، مطمئن شوید که فرمت‌های مختلف فونت (مانند woff, ttf) را بارگذاری کنید تا با تمام مرورگرها سازگار باشد.


font-face@:

ویژگی font-face@ در CSS به شما این امکان را می‌دهد که فونت‌های سفارشی را به وب‌سایت خود اضافه کنید، حتی اگر آن فونت‌ها روی سیستم کاربر نصب نشده باشند. این روش معمولاً برای بارگذاری فونت از یک فایل محلی یا از یک سرویس وب فونت (مانند Google Fonts) استفاده می‌شود.

ساختار کلی font-face@:

} font-face@
/* نام دلخواه برای فونت */ ;font-family: "CustomFontName"
/* مسیر و فرمت فونت */ ;src: url("path-to-font-file") format("font-format")
/* وزن فونت (اختیاری) */ ;font-weight: normal
/* استایل فونت (اختیاری) */ ;font-style: normal
{

پارامترهای اصلی:

1.font-family:نامی که به فونت اختصاص می‌دهید و برای استفاده در CSS به آن ارجاع می‌کنید.

2.src:مسیر فایل فونت. این مسیر می‌تواند محلی یا URL باشد:
محلی: url("fonts/MyFont.woff2")
آنلاین: url("https://example.com/fonts/MyFont.woff2")

3.format:نوع فایل فونت (برای مرورگرها). مقادیر رایج:
woff2 (فشرده و بهینه‌شده)/woff (نسخه قدیمی‌تر)/ truetype یا ttf/opentype یا otf/embedded-opentype یا eot/ svg (برای مرورگرهای قدیمی)

4.font-weight و font-style:مشخص‌کننده وزن و استایل فونت (مانند عادی، بولد، ایتالیک). این مقادیر اختیاری هستند.
