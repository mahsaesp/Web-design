Tage span:

تگ <span>در HTML یک تگ Inline است که برای گروه‌بندی محتوای متنی در یک خط (بدون شکستن خط) و اعمال استایل‌ها یا ویژگی‌های خاص به آن استفاده می‌شود. این تگ برخلاف تگ‌های بلاک‌سطری مانند <div> محتوای خود را در همان خط باقی می‌گذارد و تنها برای تنظیمات جزئی یا تغییرات در یک بخش خاص از متن به کار می‌رود.

<p>این یک متن است که در آن از تگ <span>span</span> استفاده شده است.</p>

Tage from:

در HTML تگ <form> برای ساخت فرم‌های ورودی استفاده می‌شود. فرم‌ها به کاربران امکان می‌دهند داده‌ها را وارد کنند و این داده‌ها را به یک سرور ارسال کنند.

ویژگی‌های اصلی تگ:

1-url:یا مسیری که داده‌های فرم به آن ارسال می‌شود.
2-method:روشی که برای ارسال داده‌ها استفاده می‌شود (GET یا POST).
3-enctype:نوع رمزگذاری داده‌های فرم (مثلاً برای ارسال فایل از multipart/form-data استفاده می‌شود).
4-target:تعیین می‌کند پاسخ فرم در کجا نمایش داده شود (_self, _blank, _parent, _top).
5-novalidate:اگر اضافه شود، مرورگر اعتبارسنجی فرم را انجام نمی‌دهد.

Tage input:

تگ <input> یکی از پرکاربردترین تگ‌های HTML است که برای گرفتن ورودی از کاربر استفاده می‌شود. این تگ به‌طور پیش‌فرض یک تگ self-closing است، یعنی نیازی به تگ پایانی ندارد. نوع ورودی مشخص می‌کند که این تگ چگونه عمل کند و چه نوع داده‌ای از کاربر بگیرد.


ویژگی‌های اصلی تگ input:
1-type: تعیین نوع ورودی (مانند text, password, email, checkbox و غیره).br 2-name:نام فیلد برای شناسایی آن در زمان ارسال داده‌ها.
3-value:مقدار پیش‌فرض یا مقدار فعلی فیلد.
4-placeholder:متن راهنمایی که داخل ورودی نمایش داده می‌شود.
5-required:اگر اضافه شود، فیلد باید قبل از ارسال فرم پر شود.
6-disabled:غیرفعال کردن ورودی.
7-readonly:فقط خواندنی کردن ورودی.
8-maxlengthوminlength: تعیین حداقل و حداکثر تعداد کاراکترها.
9-pattern: اعمال الگوی خاص برای ورودی.
10-autocomplete:فعال/غیرفعال کردن قابلیت تکمیل خودکار.

Attribut id:

در HTML، ویژگی id برای مشخص کردن یک شناسه یکتا (Unique Identifier) برای یک عنصر استفاده می‌شود. مقدار این ویژگی باید در کل سند یکتا باشد و نمی‌تواند برای دو عنصر مشابه استفاده شود.

</p>این متن قابل تغییر است.<p id="text">



Attribut for:

ویژگی for در HTML به‌طور خاص در تگ <label> استفاده می‌شود و عنصری را که این برچسب به آن اشاره دارد، مشخص می‌کند. این ویژگی باعث می‌شود که کلیک روی برچسب (label) باعث فعال شدن یا فوکوس کردن روی عنصر مرتبط شود. مقدار for باید برابر با مقدار ویژگی id عنصری باشد که برچسب به آن اشاره می‌کند. معمولاً برای عناصر فرم مانند <input>, <textarea>, و <select> استفاده می‌شود.

</label>نام کاربری:<label for="username">
<input type="text" id="username" name="username">



Tage textarea:

تگ <textarea> در HTML برای ایجاد یک ناحیه متنی چند خطی استفاده می‌شود که کاربر می‌تواند در آن متن وارد کند. برخلاف تگ <input type="text"> فقط یک خط ورودی فراهم می‌کند،که <textarea> برای متن‌های طولانی مناسب است.
ویژگی‌های مهم:
rows:تعداد خطوط (ارتفاع) ناحیه متنی.
cols:تعداد کاراکترها در هر خط (عرض ناحیه متنی).

<textarea name="message" rows="4" cols="50"></textarea>



Tage select:

تگ <select> در HTML برای ایجاد یک منوی کشویی استفاده می‌شود که به کاربر امکان می‌دهد یک یا چند گزینه را انتخاب کند. این تگ معمولاً همراه با تگ‌های داخلی <option> برای تعریف گزینه‌های موجود استفاده می‌شود.

ویژگی‌های مهم:
1-name:نام منوی کشویی که هنگام ارسال فرم استفاده می‌شود.
2-id:شناسه یکتا برای اتصال به برچسب <label<یا استفاده در CSS و JavaScript.
3-multiple:به کاربر امکان انتخاب چند گزینه را می‌دهد.
4-size:تعداد گزینه‌هایی که همزمان نمایش داده می‌شوند
5-required:کاربر باید حداقل یک گزینه را انتخاب کند.
6-disable:غیرفعال کردن منو.
