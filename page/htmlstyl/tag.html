Camel Case:در این حالت فقط حروف اول کوچک است.

Paskal Case: در این حالت تمام حروف اول کلمات بزرگ است

Kebab Case:در این حالت بین حروف از دش (-) استفاده میشود .

Snak Case:دراین حالت بین حروف از آندرلاین(_) استفاده میشود.


آشنایی با تگ ها در HTML:



تگ های <h1> الی <h6>:

تگهای که برای heding و عنوان محتواست از h1الیh6 ارزشش کمتر میشه. زبان HTML زبان معناگرایی است که هر تگ خلاصه یک اسمی است. تگ h1 فقط درونش متن میپذیره و پاراگراف نمی پذیره. تگ <p> </p>:
این تگ پاراگراف است .

تگ < br>:

این تگ یک تگ مفرد است و برای شکستن خط استفاده می شود.

تگ <pre> </pre>:

این تگ مخفف Preformatted texاست که درآن متن را فرمت دهی میشود و هر متنی وارد شود به همان صورت نمایش داده می شودو در این تگ به دلیل فرمت دهی شده مجاز به استفاده تگ دیگری نمی باشیم.

کامنت گذاری:

کامنت به معنی قطعه کدی است که اجرا نمی شود و برای توضیحات بیشتر در ج می شود نحوه کامنت گذاری در HTML به صورت زیر می باشد

<--متن و توضیح مورد نظر!-->


تگ <b> </b>:

این تگ نوشته و متن ها را بولد میکند.


تگ <i> </i>:

این تک نوشته و متن های ما را ایتالیک(مورب) می کند.


تگ <del> </del>:

برای خط زدن یا خط انداختن متن و کلمات استفاده می شود و تگ برعکس آن تگ <ins> </ins>می باشد.


تگ <abbr> </abbr>:

از این تگ برای تعریف اختصارات وقتی موس روی بخشی قرار میگیره استفاده می شود.


تگ <mark> </mark>:

این تگ برای هایلات و مارک کردن یه بخش یا یک رشته استفاده می شود.


تگ <small> </small>:

این تگ برای کوچک کردن یک خط یا حتی یک کلمه استفاده می شود.


تگ <sup> </sup>:

تگ برای زمانی است که می خواهیم متن به صورت بالانویس نمایش داده شود و بیشتر در معادلات ریاضی، توان ها و نمادها استفاده می شود.


تگ <sub> </sub>:

تگ برای زمانی است که می خواهیم متن به صورت پایین نویس نمایش داده شود و بیشتر در معادلات ریاضی، توان ها و نمادها استفاده می شود.
مثال تگ های بالا [x4+log2]

تگ <ul> </ul>:

تگی است که برای ایجاد یک لیست نامرتب از آن استفاده می شود و یک تگ دیگر بنام <li> </li> را درون خود دارد


تگ <ol> </ol>:

تگی است برای ایجاد یک لیست مرتب از آن استفاده میشود و مانند تگ ul تگ li را درون خود دارد. این تگ یک ویژگی بنام typ به خودش میگیره که میتونیم مقدار اعداد و حروف رو مشخص کنیم.
  • example1
  • example2
  • example3
    1. example1
    2. example2
    3. example3
تگ <hr>:

یک تگ مفرد است که برای انداختن یک خط عمودی کامل بین بخش های صفحه استفاده میشود.


تگهای معناگرا در HTML:



تگ <strong> </strong> و <em> </em>:

این تگها، تگهای معنا گرایی است که برای موتورهای جستجو و ابزارهای صفحه خوان است. تگ <b> معنی خاصی برای موتور جستجو ندارد ولی وقتی از تگ <strong> استفاده می شود همان کار بولد کردن را انجام میدهد ولی موتور جستجو متوجه می شود این عبارت یک عبارت کلیدی در صفحه است که اهمیت دارد. و جزو کلمات کلیدی صفحه در نظر می گرید. و یا تگ <em> کار تگ <i> را انجام میدهد و در ظاهر تفاوتی ندارد ولی عبارت شما را برای موتور جستجو تاکیدی می کند.


ویژگی ها(ATTRIBUT):




تگ ها معمولا یک ویژگی به خود می گیرند که ATTRIBUT نام دارد که با گرفتن این ویژگی ها قابلیتی به تگ اضافه می شود و این ویژگی ها دو حالت دارند یا مقدار دارند و یا بی مقدار هستند 
                <"مقدار مد نظر":tag ATTRIBUT name> که این یک ویژگی است که مقدار پذیر است
                <tag ATTRIBUT name> که این یک ویژگی است که بی مقداراست
Attribut lang:

در HTML برای مشخص کردن زبان محتوای یک عنصر استفاده می‌شود. این ویژگی به مرورگرها، موتورهای جستجو، و ابزارهای کمکی مانند صفحه‌خوان‌ها کمک می‌کند تا محتوا را بهتر پردازش کنند.

Attribut dir:

ویژگی dir در HTML برای تعیین جهت متن استفاده می‌شود. این ویژگی به مرورگر می‌گوید که محتوای یک عنصر باید از چپ به راست (LTR) یا از راست به چپ (RTL) نمایش داده شود.

Tage Meta:

تگ <meta> در HTML برای تعریف اطلاعات متا (Metadata) درباره‌ی صفحه وب استفاده می‌شود. این اطلاعات برای مرورگرها، موتورهای جستجو و سایر سرویس‌ها به‌کار می‌رود و شامل توضیحات، کدگذاری کاراکترها، تنظیمات ویوپورت و موارد دیگر است. تگ‌های <meta> معمولاً در بخش <head> قرار می‌گیرند.

Tage Meta Charset:

تگ <meta charset> برای تعریف نوع کدگذاری کاراکترها (Character Encoding) استفاده می‌شود. این تگ به مرورگر می‌گوید که چگونه کاراکترهای موجود در صفحه را تفسیر کند. استفاده صحیح از این تگ برای نمایش درست محتوای متنی، به‌ویژه برای زبان‌هایی با کاراکترهای خاص مثل فارسی، عربی، چینی و غیره، ضروری است.
<meta charset="UTF-8">
رایج ترین و پرکاربردترین کد گذاری utf-8 است. زیرااز تمام زبان‌ها (مانند فارسی، عربی، انگلیسی، چینی و ...) پشتیبانی می‌کند.

Tage meta viewport:

تگ <meta name="viewport">در HTML برای تنظیم نحوه نمایش صفحه وب در دستگاه‌های مختلف استفاده می‌شود. این تگ به‌ویژه برای طراحی وب‌سایت‌های ریسپانسیو (Responsive) که در موبایل و دسکتاپ به‌درستی نمایش داده می‌شوند، ضروری است.
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Tage dl:

تگ <dl> در HTML برای ایجاد یک لیست توضیحی یا لیست تعریف‌ها (Definition List) استفاده می‌شود. این نوع لیست معمولاً شامل مجموعه‌ای از اصطلاحات و توضیحات مربوط به آن‌ها است. تگ <dl> به همراه تگ‌های زیر استفاده می‌شود:
1- <dt> : برای تعریف یک اصطلاح یا عنوان.مخفف(Definition Term)
2-<dd>: برای ارائه توضیحات یا شرح مربوط به اصطلاح.مخفف (Definition Description)
<dl>
<dt>HTML</dt>
<dd>یک زبان نشانه‌گذاری برای طراحی صفحات وب.</dd>
<dl/>

Tage q & blockquote:

Tage a:

Tage img:

تگ <img> در HTML برای نمایش تصاویر استفاده می‌شود. این تگ یک عنصر تک‌پایانی است، یعنی به تگ بسته نیازی ندارد.
ویژگی‌های مهم:
1-src (Source): مسیر یا آدرس تصویر.
2-alt (Alternative Text):توضیحات جایگزین در صورتی که تصویر بارگذاری نشود استفاده از این ویژگی برای دسترس‌پذیری و سئو مهم است.

3-width و height: تنظیم عرض و ارتفاع تصویر به پیکسل یا درصد.
<"img src="image.jpg" alt="in yek tasvire nemone ast" width="200" height="150>
4-title: متنی که هنگام قرار گرفتن نشانگر ماوس روی تصویر نمایش داده می‌شود.
<"img src="image.jpg" alt="pic" title="in tek tasvir ast>
5-loading: برای بارگذاری تنبل (Lazy Loading) استفاده می‌شود.
-lazy: تصویر تنها زمانی بارگذاری می‌شود که در دید کاربر قرار گیرد.
-eager: تصویر فوراً بارگذاری می‌شود.
<"img src="image.jpg" alt="pic" loading="lazy>
6-style: افزودن استایل مستقیم به تصویر.
<"img src="image.jpg" style="border-radius: 10px>

Tage iframe:

تگ <iframe> در HTML برای جاسازی محتوای وب از یک منبع خارجی (یک صفحه وب یا دیگر منابع) در داخل صفحه وب شما استفاده می‌شود. این تگ به شما اجازه می‌دهد که محتوای یک صفحه وب دیگر را بدون ترک صفحه فعلی نمایش دهید. <iframe src="https://www.example.com" width="600" height="400"> </ifram>

ویژگی‌های تگ:

-src: این ویژگی آدرس URL صفحه‌ای است که می‌خواهید در داخل iframe نمایش داده شود
-width و height: این ویژگی‌ها عرض و ارتفاع iframe را تعیین می‌کنند.
-allowfullscreen: این ویژگی برای امکان فعال‌سازی حالت تمام صفحه برای محتوای داخل iframe است
-sandbox: این ویژگی محدودیت‌هایی را برای iframe اعمال می‌کند تا از حملات امنیتی جلوگیری کند. برای مثال، با تنظیم این ویژگی، می‌توانید اجرایی کردن اسکریپت‌ها یا ارسال فرم‌ها را محدود کنید.
