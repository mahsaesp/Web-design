مبانی و اصول اولیه HTML CSS SASS

HTML

(HyperText Markup Language)
زبان نشانه‌گذاری استانداردی است که برای ایجاد و ساختاردهی صفحات وب استفاده می‌شود. این زبان به کمک تگ‌ها، بخش‌های مختلف یک صفحه وب مانند متن، تصاویر، لینک‌ها و جداول را تعریف می‌کند HTML معمولاً همراه با CSS و JavaScript برای طراحی و عملکرد بهتر وب‌سایت‌ها استفاده می‌شود.

CSS

(Cascading Style Sheets)
یک زبان طراحی است که برای زیباتر کردن و استایل‌دهی به صفحات وب استفاده می‌شود. می‌توانید ظاهر و چیدمان عناصر HTML را با CSS تغییر دهید. امکان تغییر رنگ‌ها، فونت‌ها، فاصله‌ها، اندازه‌ها، حاشیه‌ها و بسیاری از ویژگی‌های دیگر را فراهم می‌کند با استفاده از css می‌توانید صفحات وب را واکنش‌گرا (Responsive) کنید.

SASS

(SyntacticallyAwesomeStylesheets)
یک پیش‌پردازنده‌ی css است که به شما اجازه می‌دهد کدهای CSS را به صورت ساختاریافته‌تر، قابل‌خواندن‌تر و قابل‌مدیریت‌تر بنویسید. با استفاده از sass می‌توانید از امکاناتی مانند متغیرها، توابع، تو در تو نوشتن کدها و ماژول‌ها استفاده کنید
